# 🗳️ Online Voting System

An **Online Voting System** developed using **HTML, CSS, JavaScript, and Bootstrap 5**. The application provides a simple voting platform where users can register, log in as a voter or administrator, cast votes for political parties, and view election results.

This project demonstrates the basic concepts of front-end web development, user authentication, voting logic, and result management using JavaScript.

---

## 📌 Project Overview

The Online Voting System is designed to simulate the voting process for a Lok Sabha election. It provides separate interfaces for voters and administrators.

- Voters can register, log in, and cast their vote.
- Administrators can log in and view election results.
- Each vote is counted and displayed instantly on the admin dashboard.

## 🔒 Current Limitations

- User data is stored only in browser memory.
- Votes are not stored permanently.
- No backend database.
- No encryption for passwords.
- No one-vote-per-user enforcement after page refresh.
- No email or OTP verification.
